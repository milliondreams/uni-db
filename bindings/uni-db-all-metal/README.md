# uni-db-all-metal

Python bindings for Uni Graph Database with **every provider + Apple acceleration**.

Apple Silicon only (macOS arm64). ORT CoreML EP for ONNX/fastembed/reranking, candle Metal kernels for mistralrs LLMs/vision/diffusion/speech. Both stacks GPU-accelerated; no host setup required.

For the full wheel matrix, see [the wheel matrix doc](https://github.com/rustic-ai/uni-db/blob/main/docs/proposals/python-wheel-matrix.md). For full documentation, see https://rustic-ai.github.io/uni-db.

```
pip install uni-db-all-metal
```
