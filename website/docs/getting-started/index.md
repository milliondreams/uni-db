## Quick Install

=== "Rust"

    ```bash
    cargo add uni-db
    ```

=== "Python"

    ```bash
    pip install uni-db
    ```

## Next Steps

Once installed, follow the [Quick Start](quickstart.md) to build your first reasoning graph. Then see the [AI Agent Skill](../guides/ai-skill.md) guide to integrate Uni into your agent.
